* 基于History + Ajax自制pjax组件,在列表查询页中配合分页实现了局域异步刷新，Ajax缓存,极大提升了用户体验;

* js严格遵循eslint要求、代码风格为面向对象、模块化开发;驼峰命名变量;熟练掌握注释语法以确保Documentation正常生成js接口文档。
